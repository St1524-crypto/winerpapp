# winerpapp

ERP / 電商 / 會員 / 推薦 / 點數 / 獎金管理系統

## 技術架構

* Vite
* React
* TanStack Start
* Supabase

## 主要功能

### ERP 系統

* 商品管理
* 訂單管理
* 庫存管理
* 會員管理
* 點數管理
* 獎金管理

### 電商系統

* B2C 線上購物
* B2B 批發下單
* 拼購活動

### 會員制度

* 免費會員
* VIP會員
* 推薦制度
* 安置制度
* 組織圖

### 獎金制度

* 日獎金
* 月獎金
* 推薦獎勵
* 復購獎勵
* 點數發放

## 安裝

```bash
npm install
```

## 開發模式

```bash
npm run dev
```

## 正式打包

```bash
npm run build
```

## 環境變數

請參考：

.env.example

## Supabase

請設定：

* VITE_SUPABASE_URL
* VITE_SUPABASE_ANON_KEY

## 部署建議

本專案為：

* Vite
* React
* TanStack Start
* Supabase

不建議使用一般共享主機直接上傳靜態檔。

建議部署：

* VPS
* Cloudflare
* Railway
* Render
* Hostinger VPS
